# GeoSafe v5.0 | National Hydro-Intelligence Platform
**Developed by: Geol. E. Rukevwe**

## 🌐 Project Overview
GeoSafe is a specialized Decision Support System (DSS) designed for groundwater exploration and borehole management across Nigeria’s six geopolitical zones. It integrates geological spatial data with real-time field auditing to minimize dry-hole risks and optimize drilling budgets.

## 🚀 Key Features
* **Hydro-Intelligence Engine:** Predictive modeling for drilling depth and SOPs based on regional geology.
* **Geophysical Layer Log:** Dynamic input for Vertical Electrical Sounding (VES) data ($ \Omega m $ and thickness).
* **Variance Control Audit:** Automatic calculation of "Predicted vs. Actual" drilling variance for QAQC.
* **Field Data Export:** Instant generation of **PDF Audit Reports** and **CSV Field Logs**.
* **Interactive GIS Mapping:** Real-time location tagging using Leaflet.js.

## 🛠️ Tech Stack
* **Frontend:** HTML5, Tailwind CSS
* **Mapping:** Leaflet.js & OpenStreetMap
* **Data Processing:** JavaScript (ES6+)
* **Reporting:** jsPDF & AutoTable
* **Environment:** Termux & Acode (Mobile-First Workflow)

## 📊 How it Works
1. **Select Location:** Choose Zone, State, and LGA.
2. **Analyze Intelligence:** Pull regional drilling SOPs and predicted depths.
3. **Log Field Data:** Enter resistivity layers encountered.
4. **Audit & Export:** Calculate variance and generate the official field report.
